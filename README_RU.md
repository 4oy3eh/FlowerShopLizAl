# 🌷 FlowerShop

Система управления сезонной продажей тюльпанов к 8 марта. Г. Измаил.

---

## Требования

| Что | Версия |
|-----|--------|
| Python | 3.11+ |
| ngrok | любая (с авторизацией) |
| ОС | Windows (WSL / Git Bash) или macOS / Linux |

---

## Первый запуск

### 1. Установить зависимости

```bash
pip install -r requirements.txt
```

### 2. Установить и авторизовать ngrok

Скачать: https://ngrok.com/download

```bash
ngrok config add-authtoken ВАШ_ТОКЕН
```

Токен берётся в личном кабинете на ngrok.com (бесплатный план достаточен).

### 3. Запустить

```bash
bash start.sh
```

Скрипт автоматически:
- загрузит начальные данные (сорта, упаковка, ленты, настройки)
- запустит Flask на `http://localhost:5000`
- откроет ngrok-туннель
- выведет публичную ссылку для команды

---

## Каждодневный запуск

```bash
bash start.sh
```

Вывод будет примерно таким:

```

📦 Загрузка начальных данных...
  tulip_varieties: already seeded
  ...
🚀 Запуск Flask на порту 5000...
🌐 Открываем ngrok туннель...

==============================================
  FlowerShop запущен!
----------------------------------------------
  Локально:  http://localhost:5000
  Команде:   https://xxxx-xx-xx.ngrok-free.app
==============================================
```

Скопируйте ссылку `Команде:` и отправьте в чат.

---

## Раздать ссылку команде

1. Запустите `bash start.sh`
2. В выводе найдите строку `Команде: https://...`
3. Отправьте эту ссылку в общий чат (Telegram / Viber)
4. Ссылка работает, пока запущен `start.sh` на вашем компьютере

> **Важно:** каждый раз при перезапуске ngrok выдаёт новую ссылку.
> При бесплатном тарифе ссылка меняется при каждом запуске.

## Добавить на домашний экран iPhone (Safari)

Чтобы приложение открывалось как нативное — без адресной строки:

1. Откройте ссылку FlowerShop в **Safari** на iPhone
2. Нажмите кнопку **«Поделиться»** (квадрат со стрелкой вверх) внизу экрана
3. Прокрутите меню вниз → выберите **«На экран "Домой"»**
4. Нажмите **«Добавить»**

Иконка FlowerShop появится на домашнем экране. При открытии — полноэкранный режим, как у обычного приложения.

> Повторить для каждого телефона в команде (оператор, сборщик, курьер).

## Роли и ссылки

| Роль | Основная ссылка |
|------|----------------|
| Оператор | `/orders/new` — принять заказ |
| Сборщик | `/orders?status=confirmed` — список на сборку |
| Курьер | `/courier/<route_id>` — маршрут дня |
| Администратор | `/` — дашборд |

---

## Бэкапы

База данных автоматически копируется каждый час в `data/backups/`.
Хранятся последние 7 дней. Файлы вида: `flower_shop_20250307_1400.db`.

Ручной бэкап — просто скопируйте `data/flower_shop.db` куда угодно.

---

## Остановка

Нажмите `Ctrl+C` в терминале со `start.sh`.
Flask и ngrok остановятся автоматически.

## Структура проекта

```
app.py              — точка входа
config.py           — настройки
database/           — схема БД, начальные данные, подключение
routes/             — Flask blueprints
services/           — бизнес-логика
templates/          — Jinja2 шаблоны (mobile-first)
static/             — статика
data/               — flower_shop.db + бэкапы (gitignored)
docs-ai/            — архитектура и бизнес-правила (только для чтения)
```
